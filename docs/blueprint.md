# **App Name**: SpiceMeal Hub

## Core Features:

- Menu Showcase: Display the restaurant's menu with dishes, descriptions, and prices. Allow users to filter by category (e.g., appetizers, main courses, desserts).
- Order Customization: Enable users to customize their orders (e.g., specify spice level, add extra ingredients) with real-time price updates. Generative AI suggests customization options based on popular choices and dietary restrictions acting as a helpful tool.
- Real-time Availability: Show real-time availability of dishes. Mark dishes as 'sold out' when they are unavailable.
- Delivery Service Display: Clearly outline the restaurant's delivery services, including delivery areas, fees, and estimated delivery times.
- Order Tracking: Provide a real-time order tracking feature, allowing customers to see the status of their order from preparation to delivery.  An option to send automatic SMS/email notifications with order updates.
- Contact Information: Prominently display contact information, including phone number, email address, and a contact form for inquiries.
- Responsive Design: Ensure the website is fully responsive and accessible on both desktop and mobile devices, providing a seamless user experience across all platforms.

## Style Guidelines:

- Primary color: Saffron (#F4C430) for a warm, inviting feel that represents spices.
- Background color: Light Beige (#F5F5DC) for a subtle and appetizing backdrop.
- Accent color: Terracotta (#E2725B) to highlight important information and calls to action.
- Body font: 'PT Sans', a sans-serif with a modern yet approachable feel. Headline font: 'Playfair', a modern serif, for headlines.
- Use clear, minimalist icons for menu categories, delivery options, and contact information.
- Maintain a clean, well-organized layout with clear sections for menu, delivery information, and contact details.
- Use subtle hover animations on menu items to provide visual feedback to users.
